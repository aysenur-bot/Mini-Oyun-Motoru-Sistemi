# C - Mini Oyun Motoru

Bu projede C seçeneği olan Mini Oyun Motoru seçilmiştir.

Oyun içerisinde Player, Enemy ve Item gibi nesneler bulunmaktadır.
Başlangıçta tüm davranışlar tek bir sınıf içinde ve if-else yapılarıyla kontrol edilecektir.

Bu yapı ilerleyen aşamalarda design pattern kullanılarak iyileştirilecektir.

# Mini Game Engine

## What it does
Simple mini game engine simulation.

## Patterns Used
- Factory Method
- Facade
- Adapter
- Strategy
- Observer
